# OpenGL Boilerplate

Minimal OpenGL starter project using:

- GLFW
- GLEW

Designed for quickly bootstrapping small graphics, game dev, or rendering experiments in C/C++.

---

## Features

- OpenGL context setup
- GLFW window creation
- GLEW initialization
- Basic render loop
- Simple project structure

---

## Dependencies

Make sure the following libraries are installed on your system:

- GLFW
- GLEW
- OpenGL

### Debian / Ubuntu

```bash
sudo apt install libglfw3-dev libglew-dev
